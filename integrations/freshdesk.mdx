# Freshdesk Integration Guide

## Overview
Connect Rapida with Freshdesk to integrate with this ticketing system, AI-driven insights, customizable knowledge base, and analytics tools.

## Integration Process

### Step 1: Access Freshdesk
- After clicking the Freshdesk integration box in Rapida, you'll be redirected to Freshdesk
- Log in to your Freshdesk account if not already logged in

### Step 2: Add Credentials
- Enter your Freshdesk subdomain and API key
- You can find your API key in your Freshdesk profile settings

### Step 3: Approve Authorization
- Freshdesk will show the permissions Rapida is requesting
- Review the ticket access and agent permissions
- Click "Authorize" to approve the integration

### Step 4: Confirm Connection
- You'll be redirected back to Rapida
- A confirmation message will indicate successful connection
- Your Freshdesk tickets and customer data will now be accessible in Rapida

## Permissions Granted
- Access to tickets and customer information
- Ability to create and update tickets
- Access to knowledge base articles and analytics

## Troubleshooting
- Ensure you have admin rights in your Freshdesk account
- Check if your Freshdesk plan supports API access
- Verify that you've entered the correct subdomain and API key
