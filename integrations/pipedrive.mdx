# Pipedrive Integration Guide

## Overview
Connect Rapida with Pipedrive to utilize this sales CRM and pipeline management tool for small and medium-sized businesses.

## Integration Process

### Step 1: Access Pipedrive
- After clicking the Pipedrive integration box in Rapida, you'll be redirected to Pipedrive
- Sign in to your Pipedrive account if not already logged in

### Step 2: Add Credentials
- Enter your Pipedrive account credentials
- You may need to generate an API token in your Pipedrive settings

### Step 3: Approve Authorization
- Pipedrive will display the permissions Rapida is requesting
- Review the sales data access permissions
- Click "Allow Access" to approve the integration

### Step 4: Confirm Connection
- You'll be redirected back to Rapida
- A confirmation message will indicate successful connection
- Your Pipedrive sales data will now be accessible in Rapida

## Permissions Granted
- Access to deals, contacts, and activities
- Ability to create and update sales pipeline information
- Access to sales reports and analytics

## Troubleshooting
- Ensure you have admin access to your Pipedrive account
- Check if you've entered the correct API token
- Verify that your Pipedrive plan supports API integrations
