# Zendesk Integration Guide

## Overview
Connect Rapida with Zendesk to leverage this self-service customer support platform for enterprises.

## Integration Process

### Step 1: Access Zendesk
- After clicking the Zendesk integration box in Rapida, you'll be redirected to Zendesk
- Sign in to your Zendesk account if not already logged in

### Step 2: Add Credentials
- Enter your Zendesk subdomain and admin credentials
- You may need to create an API token in your Zendesk admin settings

### Step 3: Approve Authorization
- Zendesk will display the permissions Rapida is requesting
- Review the ticket access and agent permissions
- Click "Allow" to authorize the integration

### Step 4: Confirm Connection
- You'll be redirected back to Rapida
- A success message will confirm the integration is complete
- Your Zendesk support tickets will now be accessible within Rapida

## Permissions Granted
- Access to support tickets and customer data
- Ability to create and update tickets
- Access to user and organization information

## Troubleshooting
- Ensure you have admin access to your Zendesk account
- Check if your Zendesk plan includes API access
- Verify that you've entered the correct subdomain and credentials
