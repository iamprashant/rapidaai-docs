# Azure OpenAI

Azure OpenAI provides REST API access to OpenAI's powerful language models including GPT-4, integrated with Microsoft Azure's security and enterprise features.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **Azure-OpenAI** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **Azure-OpenAI**
   - Enter a **Key Name** (e.g., "Azure-GPT4" or "AzureOpenAI-Prod")
   - Enter your **API Key** from Azure OpenAI Service
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Log in to the [Azure Portal](https://portal.azure.com/)
2. Navigate to your Azure OpenAI resource
3. Go to "Keys and Endpoint" section
4. Copy either Key1 or Key2 to use as your API key

## Additional Configuration

For Azure OpenAI, you may also need to specify:
- Endpoint URL
- Deployment name
- API version

These can be configured in advanced settings after the initial setup.

## Supported Models

- GPT-4
- GPT-3.5 Turbo
- Embeddings
- DALL-E
- Other OpenAI models available through Azure
