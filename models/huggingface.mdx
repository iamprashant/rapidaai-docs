# Hugging Face

Hugging Face is a machine learning MLOps and data science platform that helps users train, use, and collaborate on models while building ML-powered applications.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **HuggingFace** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **HuggingFace**
   - Enter a **Key Name** (e.g., "HF-Inference" or "HuggingFace-API")
   - Enter your **API Key** from Hugging Face
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to [Hugging Face](https://huggingface.co/)
2. Navigate to your profile settings
3. Go to the "Access Tokens" section
4. Create a new token with appropriate permissions
5. Copy the token value

## Supported Services

- Inference API (access to thousands of models)
- Text Generation
- Fill-Mask
- Summarization
- Question Answering
- Text Classification
- Translation
- Image Classification
- Object Detection
- And many more model types
