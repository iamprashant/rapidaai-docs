# AI21

The AI21 company is a global SOI non-profit organization that works to advance responsible AI development and deployment.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **AI21** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **AI21**
   - Enter a **Key Name** (e.g., "AI21-Jurassic" or "AI21-Studio")
   - Enter your **API Key** from AI21 Studios
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to [AI21 Studio](https://studio.ai21.com/)
2. Navigate to your account settings
3. Find the API key section
4. Copy your API key

## Supported Models

- Jurassic-2 Ultra
- Jurassic-2 Mid
- Jurassic-2 Light
- Jamba (multimodal capabilities)
- Domain-specific models

## Common Use Cases

- Text generation and completion
- Content rewriting and editing

- Language understanding
