# AWS Bedrock

AWS Bedrock provides foundational models for machine learning, designed for a variety of AI applications with enterprise-level security and scalability.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **AWS-Bedrock** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **AWS-Bedrock**
   - Enter a **Key Name** (e.g., "AWS-Bedrock-Prod" or "Bedrock-Claude")
   - Enter your **AWS Access Key** and **Secret Key**
4. Click **Configure** to save your settings

## Obtaining AWS Credentials

1. Log in to the [AWS Management Console](https://aws.amazon.com/console/)
2. Navigate to IAM (Identity and Access Management)
3. Create a new user or use an existing one
4. Generate access keys for programmatic access
5. Ensure the user has appropriate permissions for Bedrock

## Required Information

For AWS Bedrock, you'll need:
- AWS Access Key ID
- AWS Secret Access Key
- AWS Region (where Bedrock is available, e.g., us-east-1)

## Supported Models

Through AWS Bedrock, you can access models from:
- Anthropic (Claude)
- AI21 Labs (Jurassic)
- Cohere
- Stability AI
- Amazon Titan
- And other foundation models available on AWS Bedrock
