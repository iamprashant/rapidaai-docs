# Mistral

Mistral specializes in creating fast, secure, open-source large language models (LLMs) that balance performance with efficiency.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **Mistral** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **Mistral**
   - Enter a **Key Name** (e.g., "Mistral-Medium" or "Mistral-Large")
   - Enter your **API Key** from Mistral AI
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to the [Mistral AI Platform](https://console.mistral.ai/)
2. Navigate to the API Keys section
3. Create a new API key
4. Copy the key value
