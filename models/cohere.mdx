# Cohere

Cohere offers a smaller and faster version of language models with almost as much capability, ideal for efficient AI applications.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **Cohere** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **Cohere**
   - Enter a **Key Name** (e.g., "Cohere-Command" or "Cohere-Embed")
   - Enter your **API Key** from Cohere
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to the [Cohere Platform](https://dashboard.cohere.com/)
2. Navigate to the API Keys section
3. Create a new API key or use the existing one
4. Copy the key value

## Supported Models

- Command (text generation)
- Command Light
- Command R (multilingual)
- Embed (embeddings)
- Rerank (semantic search)

## Common Use Cases

- Text generation
- Semantic search
- Document classification
- Content moderation
- Embeddings generation
