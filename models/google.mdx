# Google

Google AI offers access to Google's latest family of large language models, including Gemini and PaLM, with state-of-the-art capabilities.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **Google** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **Google**
   - Enter a **Key Name** (e.g., "Google-Gemini" or "Google-AI")
   - Enter your **API Key** from Google AI Studio or Google Cloud
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to [Google AI Studio](https://makersuite.google.com/) or [Google Cloud Console](https://console.cloud.google.com/)
2. For Google AI Studio:
   - Navigate to the API keys section
   - Create a new API key
3. For Google Cloud:
   - Enable the Vertex AI API
   - Create credentials for API access
4. Copy the API key value

- Gemini Flash
