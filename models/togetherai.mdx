# TogetherAI

Together AI is a research-driven artificial intelligence company that contributes leading-edge foundation models to the open-source community.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **TogetherAI** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **TogetherAI**
   - Enter a **Key Name** (e.g., "Together-Prod" or "Together-LLM")
   - Enter your **API Key** from Together AI
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to [Together AI](https://www.together.ai/)
2. Navigate to your account dashboard
3. Find the API Keys section
4. Create a new API key
5. Copy the key value
