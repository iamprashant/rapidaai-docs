# VoyageAI

VoyageAI is a cutting-edge AI platform offering comprehensive solutions for various AI applications with specialized embedding models.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **VoyageAI** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **VoyageAI**
   - Enter a **Key Name** (e.g., "Voyage-Embeddings" or "VoyageAI-Main")
   - Enter your **API Key** from VoyageAI
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to [VoyageAI](https://www.voyageai.com/)
2. Navigate to your account dashboard
3. Look for the API section
4. Generate a new API key
5. Copy the key value
