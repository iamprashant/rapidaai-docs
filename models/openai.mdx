# OpenAI

OpenAI has broad general knowledge and domain expertise, allowing it to handle a wide range of tasks from text generation to creative content creation.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **OpenAI** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **OpenAI**
   - Enter a **Key Name** (e.g., "OpenAI-GPT4" or "OpenAI-Production")
   - Enter your **API Key** from OpenAI
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to your [OpenAI account](https://platform.openai.com/)
2. Navigate to the [API keys page](https://platform.openai.com/account/api-keys)
3. Create a new secret key
4. Copy the key value (note that it typically starts with "sk-")

## Supported Models

- GPT-4 and GPT-4 Turbo
- GPT-3.5 Turbo
- DALL-E 3 (image generation)
- Whisper (speech-to-text)
- Embeddings models

## Common Use Cases

- Text generation and completion
- Content creation and editing
- Conversational AI
- Code assistance
- Knowledge extraction
