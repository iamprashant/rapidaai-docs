# DeepInfra

Deep Infra offers cost-effective, scalable, easy-to-deploy, and production-ready machine learning infrastructure for AI applications.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **DeepInfra** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **DeepInfra**
   - Enter a **Key Name** (e.g., "DeepInfra-Main" or "DeepInfra-LLM")
   - Enter your **API Key** from DeepInfra
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to [DeepInfra](https://deepinfra.com/)
2. Navigate to your account dashboard
3. Find the API section
4. Generate or copy your API key

## Supported Models

DeepInfra hosts a variety of open-source models including:
- Mixtral
- Llama 2
- CodeLlama
- Falcon
- Stable Diffusion
- And many other optimized open-source models

## Common Use Cases

- Text generation and completion
- Code generation
- Image generation
- Cost-effective inference
- Scalable AI deployment
