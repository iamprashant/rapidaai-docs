# Deepgram

Deepgram provides cutting-edge speech recognition services using advanced AI technologies for high-accuracy voice-to-text conversion.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **Deepgram** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **Deepgram**
   - Enter a **Key Name** (e.g., "Deepgram-Speech" or "Deepgram-Recognition")
   - Enter your **API Key** from Deepgram
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to [Deepgram](https://deepgram.com/)
2. Navigate to your account dashboard or settings
3. Find the API or Developer section
4. Generate a new API key
5. Copy the key value
