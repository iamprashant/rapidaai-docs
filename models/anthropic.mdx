# Anthropic

Anthropic offers Claude, a versatile AI assistant model that can handle a range of tasks with high sensitivity and safety measures.

## Configuration Steps

1. Navigate to the **Providers and Models** section in your Rapida.ai dashboard
2. Locate and click on the **Anthropic** card
3. In the "Create provider credential" popup:
   - The provider will be pre-selected as **Anthropic**
   - Enter a **Key Name** (e.g., "Anthropic-Production" or "Claude-Main")
   - Enter your **API Key** from Anthropic
4. Click **Configure** to save your settings

## Obtaining an API Key

1. Sign up or log in to your [Anthropic Console](https://console.anthropic.com/)
2. Navigate to the API Keys section
3. Create a new API key or use an existing one
4. Copy the key value (note that it typically starts with "sk-ant-")

## Using Anthropic Models

Once configured, you can access Claude models through Rapida.ai for:
- Text generation
- Content summarization
- Question answering
- Creative writing
- Code assistance

## Supported Models

- Claude 3 Opus
- Claude 3 Sonnet
- Claude 3 Haiku
- Claude 2
- Claude Instant
