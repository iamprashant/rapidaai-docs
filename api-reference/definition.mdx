### Error

  Error information, if the operation failed. Present in every response object.

  <ResponseField name="errorCode" type="string (uint64)">
    Stable, machine-readable numeric code. Represented as a string in JS/TS; parse only if you need numeric comparisons.
  </ResponseField>
  <ResponseField name="errorMessage" type="string">
    Developer-oriented message with diagnostic details. Avoid showing directly to end-users.
  </ResponseField>
  <ResponseField name="humanMessage" type="string">
    Human-friendly description safe for display to end-users. Localize in your app if needed.
  </ResponseField>



### Assistant Definition

The `AssistantDefinition` object should be configured as follows:

<CodeGroup>

```ts TypeScript
import { AssistantDefinition } from 'rapida-nodejs';

const assistant = new AssistantDefinition();
assistant.setAssistantid(assistantId);
assistant.setVersion('latest');
```

```javascript Node.js
const { AssistantDefinition } = require('rapida-nodejs');

const assistant = new AssistantDefinition();
assistant.setAssistantid(assistantId);
assistant.setVersion('latest');
```

```python Python
from rapida_sdk import AssistantDefinition

assistant = AssistantDefinition()
assistant.setAssistantid(assistant_id)
assistant.setVersion("latest")
```

```go Go
import rapida "github.com/rapidaai/rapida-go"

assistant := rapida.NewAssistantDefinition()
assistant.SetAssistantid(assistantID)
assistant.SetVersion("latest")
```

</CodeGroup>

| Method | Parameter | Description |
|--------|-----------|-------------|
| `setAssistantid()` | `string` | Sets the unique identifier for the assistant |
| `setVersion()` | `string` | Sets the version of the assistant (use 'latest' for current version) |

### Connection Configuration

Use `ConnectionConfig` to authenticate your SDK calls.

<CodeGroup>

```ts TypeScript
import { ConnectionConfig } from 'rapida-nodejs';

const clientCfg = new ConnectionConfig();
clientCfg.withAuth(
  ConnectionConfig.WithSDK({
    apiKey: process.env.RAPIDA_API_KEY!,
  })
);
```

```go Go
package main

import (
  "os"
  sdk "github.com/rapidaai/rapida-go"
)

func main() {
  cfg := sdk.NewConnectionConfig()
  cfg.WithAuth(sdk.WithSDK(sdk.SDKOptions{
    ApiKey: os.Getenv("RAPIDA_API_KEY"),
  }))
}
```

```python Python
from rapida_sdk import ConnectionConfig, WithSDK
import os

connection_config = ConnectionConfig()
connection_config.with_auth(WithSDK(api_key=os.getenv("RAPIDA_API_KEY")))
```

</CodeGroup>
