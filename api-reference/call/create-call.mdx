title: "Create Phone Call"
---

Creates an outbound phone call to the specified number using your configured assistant.

<Info>
  For full setup and authentication options, see the Installation guide: [/api-reference/introduction](/api-reference/introduction).
</Info>

<ParamField body="clientCfg" type="ConnectionConfig" required>
  Configuration for the client connection.
</ParamField>

<ParamField body="assistant" type="AssistantDefinition" required>
  Assistant configuration object.
</ParamField>

<ParamField body="toNumber" type="string" required>
  Destination phone number in E.164 format (for example, +14155550123).
</ParamField>

<ParamField body="fromNumber" type="string">
  Caller ID phone number to originate the call from (E.164 format). If omitted, the system default will be used.
</ParamField>

<ParamField body="args" type="Map<string, google_protobuf_any_pb.Any>">
  Additional arguments passed to the call as a key-value map.
</ParamField>

<ParamField body="options" type="Map<string, google_protobuf_any_pb.Any>">
  Call options for runtime behavior and configuration.
</ParamField>

<ParamField body="metadata" type="Map<string, google_protobuf_any_pb.Any>">
  Arbitrary metadata to associate with the call for tracking and analytics.
</ParamField>

### Usage

<CodeGroup>

```ts React
import {
  ConnectionConfig,
  CreatePhoneCall,
  AssistantDefinition,
} from 'rapida-react';

// assistant definition
const assistant = new AssistantDefinition();
assistant.setAssistantid(<ASSISTANT_ID>);
assistant.setVersion('latest');


// check authentication object
let connectionCfg = ConnectionConfig.DefaultConnectionConfig(
    ConnectionConfig.WithSDK({ ApiKey: process.env.RAPIDA_API_KEY })
);

CreatePhoneCall(connectionCfg, assistant, "<PHONE_NUMBER>")
.then(x => {
    if (x.getSuccess()) {
        console.log("call created succssfully")
    } else {
        let err = x.getError();
        if (err?.getHumanmessage()) {
            console.log(err?.getHumanmessage());
        }
    }
})
.catch(x => {
    // network error occired
    console.dir(x);
});
```

### Response

The `CreatePhoneCall` request returns a `CreatePhoneCallResponse` with the following fields:

| Field | Type | Description |
|-------|------|-------------|
| `code` | `number` | Numeric status code for the operation |
| `success` | `boolean` | Indicates whether the operation was successful |
| `data` | `AssistantConversation`, `undefined` | When successful, contains per-request results (may be undefined) |
| `error` | `Error` | When unsuccessful, contains error information (may be undefined) |
