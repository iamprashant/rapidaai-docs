---
title: 'Get Assistant'
---

Retrieves detailed information about a specific assistant using the Rapida API.

<Info>
For more authentication options, see: [https://doc.rapida.ai/api-reference/authentication](https://doc.rapida.ai/api-reference/authentication)
</Info>

### Parameters

<ParamField body="connectionConfig" type="ConnectionConfig" required>
Configuration for the client connection.
</ParamField>

<ParamField body="request" type="GetAssistantRequest" required>
<Expandable>
<ParamField body="id" type="uint64" required>
The unique identifier of the assistant to retrieve.
</ParamField>
<ParamField body="assistantProviderModelId" type="uint64" optional>
Optional identifier for the assistant provider model. Use utils.Ptr() to set this value.
</ParamField>
</Expandable>
</ParamField>

### Usage

<CodeGroup>

```go Go
package main

import (
	"context"
	"fmt"
	"os"

	clients "github.com/rapidaai/rapida-go/rapida/clients"
	rapida_proto "github.com/rapidaai/rapida-go/rapida/clients/protos"
	connections "github.com/rapidaai/rapida-go/rapida/connections"
	"github.com/rapidaai/rapida-go/rapida/utils"
)

func main() {
	// Configure client connection using RAPIDA_PROJECT_CREDENTIAL environment variable
	connectionConfig := connections.
		DefaultconnectionConfig(connections.WithSDK(os.Getenv("RAPIDA_PROJECT_CREDENTIAL")))

	// Retrieve assistant information
	response, err := clients.GetAssistant(context.Background(), connectionConfig, &rapida_proto.GetAssistantRequest{
		Id: 45678903456789,
		// Optional: Specify assistant provider model ID
		AssistantProviderModelId: utils.Ptr(uint64(3476544567890987654)),
	})

	if err != nil {
		fmt.Printf("Error while making call using rapida: %+v\n", err)
		return
	}

	// Print the response
	fmt.Printf("Rapida calling response: %+v\n", response)
}
```

</CodeGroup>
