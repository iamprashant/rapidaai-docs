## Step 5:

1. Go to Phone Call Settings: Open the “Phone Call” section under “Deployment”.
2. Name of Agent: Enter the agent's name in the Name field.
3. Role: Specify the agent's role in the Role field.
4. Communication Tone: Select the tone in the Communication Tone dropdown.
5. Expertise: Choose the agent's expertise area in Primary Expertise.
6. Opening Line of Conversation: Type the greeting in the Greeting field.
7. Message if Any Error Occurs: Input the error message in Error Message.
8. Closing Message of the Conversation: Fill in the ending message in Ending Message.

![Configure Twilio Step 5](/images/configure_twilio_5.png)

## Step 6:
### Configuration on Twilio:

Enter your Twilio credentials:

1. Twilio Account SID  
2. Twilio Auth Token  
3. Twilio Phone number used to make Inbound / Outbound calls

For detailed instructions on how to obtain these Twilio credentials, please refer to the [Twilio Credentials Guide](For detailed instructions on how to obtain these Twilio credentials, please refer to the [Twilio Credentials Guide](/assistants/twilio-credentials).


![Configure Twilio Step 6](/images/configure_twilio_6.png)

## Step 7:
##@ Speech to Text Configuration
Select the following parameters for your Speech to Text provider. 

1. Choice: Deepgram.  
2. Model. Example : Nova \-3 for Deepgram  
3. Language  
4. Encoding: Select **Mulaw** for Twilio  
5. Required sample rate.  
6. Transcript Confidence Threshold

![Configure Twilio Step 7](/images/configure_twilio_7.png)

## Step 8:
Select relevant configuration for your assistant

1. Rapida to finalize the end of speech. Disable this if you do not want automatic end of speech detection.  
2. Select correct values for Segmentation Silence Timeout. This is the duration of silence after which Rapida starts finalizing a message EOS: Based on silence and max time (1000-4000ms).  
3. Background Denoising Enabled. Default active. Disabling this will affect performance of Speech to Text transcription.  
4. User Idle Silence Timeout. After this Duration of silence, Rapida finalizes a phrase (3000-10000ms) and sends it back as audio response.  
5. Idle Message: Message that the assistant will speak when the user hasn't responded until the Idle Silence Timeout.

![Configure Twilio Step 8](/images/configure_twilio_8.png)

## Step 11:
### Final Step: Configuring Twilio Webhook URL

After obtaining your Twilio credentials, you need to configure your Twilio phone number to connect with your Rapida assistant:

1. Log in to your Twilio account at [https://www.twilio.com/login](https://www.twilio.com/login)
2. Navigate to "Phone Numbers" → "Manage" → "Active numbers"
3. Select the phone number you want to use with Rapida
4. Under "Voice & Fax" configuration:
   - Set "A CALL COMES IN" to "Webhook"
   - Use the following URL format: 

 https://assistant-01.rapida.ai/v1/talk/twilio/call/__ASSISTANT_ID__?x-api-key=RAPIDA_API_KEY

5. Replace __ASSISTANT_ID__ with your assistant's unique ID
6. Replace RAPIDA_API_KEY with your Rapida API key
7. Set the HTTP method to "HTTP POST"
8. Click "Save" to apply the changes
